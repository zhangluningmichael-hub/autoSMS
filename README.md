# AutoSMS - 安卓短信自动回复/转发程序

## 项目概述
这是一个基于 Android 的短信自动处理程序，可以根据发件人号码和短信内容关键词进行规则匹配，实现自动回复或转发功能。

## 功能特性
1. **规则匹配**：
   - 支持发件人号码匹配（支持通配符*，如138*, 10086, +86*）
   - 支持短信内容关键词匹配（多个关键词用逗号分隔）
   
2. **操作类型**：
   - 自动回复：发送预设的回复内容
   - 短信转发：将短信转发到指定号码
   
3. **规则管理**：
   - 添加、编辑、删除规则
   - 启用/禁用规则
   - 简单的配置界面

4. **权限管理**：
   - 自动请求短信相关权限
   - 服务启用/禁用控制

## 技术架构
- **语言**：Kotlin
- **架构**：MVVM + Repository
- **数据库**：Room
- **UI框架**：Material Design
- **目标SDK**：Android 16 (vivo originOS6兼容)

## 编译和运行

### 环境要求
- Android Studio 2022.3.1 或更高版本
- JDK 17 或更高版本
- Android SDK 34

### 编译步骤
1. 使用 Android Studio 打开 `/e/autoSMS` 文件夹
2. 等待 Gradle 同步完成
3. 连接 vivo 手机（确保已开启开发者选项和USB调试）
4. 点击运行按钮（▶️）或按 Shift+F10

### 手动编译
```bash
cd /e/autoSMS
./gradlew assembleDebug
```

## 使用说明

### 首次使用
1. 安装应用后打开
2. 点击"请求权限"获取短信权限
3. 点击"启用服务"开启自动处理
4. 点击"规则管理"添加规则

### 添加规则示例
1. **自动回复规则**：
   - 规则名称：快递通知回复
   - 发件人匹配：10086*
   - 内容关键词：快递,取件码,包裹
   - 操作类型：回复
   - 回复内容：已收到，谢谢！

2. **转发规则**：
   - 规则名称：重要短信转发
   - 发件人匹配：13888888888
   - 内容关键词：紧急,重要,会议
   - 操作类型：转发
   - 转发号码：13999999999

## 注意事项

### 权限说明
应用需要以下权限：
- 接收短信 (RECEIVE_SMS)
- 读取短信 (READ_SMS)
- 发送短信 (SEND_SMS)
- 读取手机状态 (READ_PHONE_STATE)
- Android 13+ 需要通知权限

### vivo originOS6 适配
- 应用已适配 Android 16 (API 34)
- 支持后台服务处理短信
- 适配最新的权限模型

### 性能优化
- 使用 Room 数据库缓存规则
- 广播接收器优化处理
- 前台服务保证后台运行

## 故障排除

### 常见问题
1. **收不到短信**：检查权限是否全部授予
2. **规则不生效**：检查规则是否启用，匹配条件是否正确
3. **应用被系统杀死**：在手机设置中将应用加入白名单

### 调试方法
1. 查看 Logcat 中的 "SmsReceiver" 和 "SmsProcessingService" 标签
2. 检查数据库中的规则是否正确保存
3. 测试时发送匹配规则的短信
